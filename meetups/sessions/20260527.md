- Installation

```
# Direct: prints nothing useful because stdio is the protocol channel
uv run python server.py

# Dev mode: launches Inspector pointed at your server, hot-reloads on edit
uv run mcp dev server.py

# Install: writes the entry into Claude Desktop's config for you
uv run mcp install server.py --name "Scratchpad"
```

- Running things

Install in claude. Claude Desktop Settings -> Developer -> Edit Config

```
  "mcpServers": {
    "scratchpad": {
      "command": "uv",
      "args": ["--directory", "/Users/sam/wd/code/bristol-ml/meetups/assets/scratchpad-mcp",
               "run", "python", "server.py"]
```
