## Install scratchpad app

Claude Desktop -> Settings -> Developer -> Edit Config -> claude_desktop_config.json add: 
  "mcpServers": {
    "scratchpad": {
      "command": "uv",
      "args": [
        "--directory",
        "/Users/sam/wd/code/bristol-ml/meetups/assets/scratchpad-mcp",
        "run",
        "python",
        "server.py"
      ]
    }
  },

With the appropriate path replacement.
