# LLD — low-level design

Lead and teammates write here freely. Contents are either append-only
retrospective or discardable scratch — nothing in this tree is
load-bearing for users or for downstream stages.

- [`stages/`](./stages/) — per-stage retrospectives (one file per
  completed stage, named `NN-slug.md`).
- `research/` — researcher agent output. Created on demand by the
  [@researcher](../../.claude/agents/researcher.md) agent.
- `reframings/` — reframer agent output. Created on demand by the
  [@reframer](../../.claude/agents/reframer.md) agent.

See [`../architecture/`](../architecture/) for structural decisions and
[`../intent/DESIGN.md`](../intent/DESIGN.md) for the project spec.
